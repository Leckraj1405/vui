HTML Beginner Guide

This always-on voice system helps students by providing HTML help, such as HTML basics, codes format, elements, attributes, and so on.

Try saying 'What is HTML' and see the magic! or even try other HTML related content such as 'attributes', 'elements'. Saying 'Open documentation' shall redirect you to another website.

Note that this VUI automatically captures voice on starting. Please allow microphone access and enable pop-ups.
